Help for Beat the Clock Game

Welcome to the "Beat the Clock Game." In this game, you are presented with a deck of 16 cards, which are face down. Each card has a symbol on it which matches one other card in the deck.

Your objective is to match each of the cards before the game clock runs out.

How to Play the Game

  • Select a card. The card flips to reveal it's symbol.
  • Remember that card's symbol, as you want to match it.
  • Select a second card, preferably one that matches the flipped over card you selected above.
  • If the two symbols match, you win points and both cards turn green. However, if the cards do not match, the cards are hidden.
  • Repeat until you match all of the cards or the game clock runs out.

The game clock starts when you click on a card. If it times out, then the game stops.

Scoring

Each game is scored. You receive points when you match a pair of cards. When you match consecutive pairs of cards, you receive a bonus. The more consecutive pairs you match, the higher the bonus points.

You lose points when missing consecutive cards.

How to Level Up

There are multiple levels in this game. If you beat the clock in the required number of moves and minimum score, you will automatically advance to the next level.
